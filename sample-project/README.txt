Si quiere probar el proyecto:

1). Descargue qtCreator en su pc
2). Cuando tenga el qtCreator abralo y en la parte superior derecha encontrará una opcion que dice file
3). Usted dará click en file y posteriormente open file or proyect
4). Entrará a la carpeta que se le mandó (sample-project) y buscará el archivo sample-proyect.pro
5). Perfecto, ya tiene el proyecto abierto!
6). Ahora usted tiene que expandir la carpeta del proyecto que se abrió dentro del qtCreator
7). Una vez hecho el paso anterior usted tendrá acceso a los Headers y a Sources (la interfaz e implementacion respectivamente)
8). Los archivos del proyecto los encontrará en una carpeta llamada src, tanto los .h como los .hpp en sus respectivas carpetas (Headers, SOurce)
9). Listo!
